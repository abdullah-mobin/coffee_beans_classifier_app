CoffeeNet
CoffeeNet is a deep learning model designed to classify coffee beans based on their roast level. It uses convolutional neural networks (CNNs) to analyze the texture and color variations in coffee bean images to determine the roast grade.
VGG
VGG is a well-known deep learning architecture used for image classification tasks. Its simple yet powerful architecture consists of several convolutional layers stacked on top of each other, making it effective in extracting features from images such as coffee beans.
ResNet
ResNet (Residual Networks) is a deep neural network architecture that uses skip connections to help with training deeper models. It has been widely used in image classification tasks and is capable of handling complex images, such as those of coffee beans, effectively.
Xception
Xception is an extension of the Inception model, which replaces the standard convolutions with depthwise separable convolutions. This makes the model more efficient while retaining its ability to handle complex image recognition tasks, such as classifying coffee beans.
